Làm Chủ CSS Grid Layout

CSS Grid Layout là một hệ thống bố cục mạnh mẽ giúp bạn xây dựng các trang web với bố cục phức tạp một cách đơn giản và dễ dàng. Hãy cùng khám phá cách làm chủ CSS Grid!

1. Giới Thiệu về CSS Grid Layout

CSS Grid Layout cung cấp một phương pháp mạnh mẽ để tạo ra các layout phức tạp mà không cần phải sử dụng nhiều lớp thụt lề hay các phương pháp hack khác.

2. Các Thành Phần Cơ Bản của Grid

Các thành phần cơ bản của Grid bao gồm: container grid, rows, columns và cells. Chúng ta có thể điều chỉnh các phần tử này bằng các thuộc tính như `grid-template-columns`, `grid-template-rows` và `grid-gap`.

2.1. Grid Container

Một Grid Layout được xác định với thuộc tính `display: grid;`. Điều này giúp các phần tử con trong container được xếp thành các hàng và cột. 

                
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
}

2.2. Các Cột và Hàng

Bạn có thể sử dụng `grid-template-columns` để xác định số lượng và kích thước các cột, và `grid-template-rows` để xác định số lượng và kích thước các hàng trong grid. 

                
.container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
    grid-template-rows: 100px 200px;
}

3. Tạo Layout với Grid

Sau khi làm quen với các thành phần cơ bản, chúng ta có thể tạo một layout đơn giản với 3 cột và 2 hàng. 

                
.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-gap: 20px;
}

.item {
    background: lightgray;
    padding: 20px;
    text-align: center;
}

4. Các Tính Năng Nâng Cao

CSS Grid còn cung cấp nhiều tính năng nâng cao như grid-template-areas, auto-placement, và việc sử dụng media queries để điều chỉnh bố cục trên các thiết bị khác nhau.
