React Hooks: Từ Cơ Bản Đến Nâng Cao

Trong bài viết này, chúng ta sẽ cùng nhau tìm hiểu về React Hooks, một tính năng mạnh mẽ của React giúp tối giản hóa việc sử dụng state và lifecycle methods trong các functional components.

1. React Hooks là gì?

React Hooks là các hàm cho phép bạn "hook vào" state và lifecycle methods của React từ trong functional components. Trước khi React ra mắt hooks, chỉ có class components mới có thể sử dụng state và lifecycle methods, nhưng giờ đây, bạn có thể làm điều đó với functional components mà không cần phải viết một class.

2. Các loại React Hooks cơ bản

useState

`useState` là một hook dùng để khai báo state trong functional component. Đây là hook cơ bản và được sử dụng rất phổ biến. 


const [count, setCount] = useState(0);

useEffect

`useEffect` là hook giúp bạn thực hiện các tác vụ phụ (side effects) trong component như gọi API, thay đổi DOM, hoặc đăng ký sự kiện. 


useEffect(() => {
    document.title = `Bạn đã nhấn ${count} lần`;
}, [count]);

3. Hooks nâng cao

Ngoài `useState` và `useEffect`, React còn cung cấp nhiều hooks nâng cao khác như `useContext`, `useReducer`, và `useCallback` để giải quyết các vấn đề phức tạp hơn trong quản lý state và tối ưu hóa hiệu suất.

4. Sử dụng custom hooks

Custom hooks là một hàm JavaScript mà bạn có thể tạo ra để tái sử dụng logic trong các component khác nhau. Custom hooks giúp mã của bạn trở nên dễ đọc và dễ duy trì hơn. 


function useCounter() {
    const [count, setCount] = useState(0);
    const increment = () => setCount(count + 1);
    return [count, increment];
}
